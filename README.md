# 🌐 Индивидуальный проект по созданию веб-сайтов

Репозиторий содержит индивидуальный проект, разработанный специально для **ГБПОУ МО «Павлово-Посадский техникум»**.

* **Официальный сайт техникума:** https://www.pp-teh.ru/

---

## 📝 О проекте

Показать реальный уровень владения инструментами программирования и веб-разработки на примере создания интерфейса и логики для сайта техникума. Это индивидуальная работа, подтверждающая мои навыки. 

Вы можете самостоятельно открыть этот проект, внести изменения и запустить его на своем локальном компьютере (инструкция по запуску приведена ниже).

## 🚀 Инструкция по запуску

Для проверки работы проекта выполните следующие шаги в терминале:

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd название-репозитория
```

### 2. Создание и активация виртуального окружения
```bash
python -m venv venv
```
## Активация для Windows (cmd / PowerShell):
```bash
venv\Scripts\activate
```

### 3. Установка зависимостей (при наличии)
```bash
pip install -r requirements.txt
```

### 4. Запуск проекта через FastAPI
```bash
fastapi dev main.py
```

### 5. Ссылка на страницу:
### http://127.0.0.1:8000
